## Быстрый старт

### Установка и запуск

1. **Клонирование репозитория**

git clone https://github.com/georgio777/slider.git
cd slider

2. **Установка зависимостей**

npm install

3. **Запуск в режиме разработки**

npm start

4. **Сборка для production**

npm run build
